# 실행 계획 (Exec Plans)

> ⚠️ **현재 미사용**
>
> 이 디렉토리는 대규모 팀 프로젝트 또는 장기 운영 서비스에서
> AI 에이전트의 작업 계획을 문서화하기 위한 공간입니다.
>
> chatflow-frontend는 1인 포트폴리오 프로젝트이므로 현재는 사용하지 않습니다.
> 팀 프로젝트로 확장 시 아래 구조를 활용하세요.

## 확장 시 구조

```
docs/exec-plans/
├── active/       # 진행 중인 작업 계획
├── completed/    # 완료된 작업 계획
└── archived/     # 오래된 작업 계획
```
